proverbs-go

A rewrite of the UNIX program "Fortune", Proverbs is the result of my first day learning Go. So far it's been amazing, the language seems about halfway between C and Python, it's definitely got a focus on readability of syntax too. Honestly the most amazing thing is not having segfaults every time I use strings; this program is pretty string heavy, yet I didn't have a single crash. And portable networking code??? That only takes like 2 functions??? Not sure if this is heresy or not. Hoping to learn more Go in the future, but for now Proverbs is gonna stay public until I make something impressive (in the works).
